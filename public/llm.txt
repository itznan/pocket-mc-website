# PocketMC

PocketMC is a free, local-first Windows desktop client designed to manage and run Minecraft Java & Bedrock servers easily, with no script writing or complex configuration required.

## Key Features

### 1. Runtime Isolation & Management
- Downloads and manages Adoptium JRE runtimes locally to prevent polluting Windows system environment variables.
- Supports local versions: Java 8, 11, 17, 21, and 25.
- Automatically prompts and downloads the correct Java version matching the Minecraft server version.
- Integrates the official PHP 8.2 runtime specifically optimized for PocketMine-MP PM5 servers.

### 2. Supported Server Software Engine Types
- **Vanilla Java**: Official Mojang server JARs, supports Minecraft 1.8.8+ up to latest.
- **Paper**: High-performance paper server builds by PaperMC, supports Minecraft 1.8.8+ up to latest.
- **Fabric**: Modular lightweight modding toolkit, supports Minecraft 1.14+ up to latest.
- **Forge**: Legacy industry-standard modding framework, installer-based downloads, Minecraft 1.8.8+ up to latest.
- **NeoForge**: Community-driven modern fork of Forge, installer-based downloads from official Maven metadata.
- **Bedrock Server (BDS)**: Official Mojang Bedrock server binary, supports stable & preview builds.
- **PocketMine-MP**: PHP-based highly extensible Bedrock server (PM5 runtime supported).

### 3. Integrated Local & Cloud Backups
- Schedules automatic background server backups with SHA-256 integrity checks.
- Pre-checks live servers: triggers RCON sync first and falls back to console save commands to avoid lock conflicts (excludes lock files like `session.lock`).
- Supports OAuth-based cloud backups directly to Google Drive, Dropbox, and OneDrive with one-click restore.

### 4. Playit.gg Tunnel Provisioning
- No router configuration or port forwarding needed.
- Provides a guided 4-step tunnel setup wizard that links the user's Playit.gg account.
- Provisions local client agents and routes local ports automatically to public numeric and human-readable tunnel addresses.

### 5. Curated Mod & Plugin Marketplaces
- Browses and installs Minecraft mods, plugins, and custom packages directly inside the app interface.
- Integrates the **Modrinth** browser (no API key needed).
- Integrates the **CurseForge** browser (requires personal API key).
- Integrates the **Poggit** marketplace for PocketMine-MP PHP plugins.
- Ingests Bedrock `.mcpack`, `.mcaddon`, and world `.zip` files, auto-parsing manifests and registering world JSON structures.

### 6. Interactive Management Tools
- Colorized real-time server log console with auto-sanitization of personal details (IP addresses, emails).
- Resource usage charts (real-time CPU and RAM logging).
- Automated pre-flight port checks to avoid address conflicts before starting server processes.
- Runs silently in the Windows System Tray with support for auto-starts and clean background operations.
- Dynamic cross-play badges that scan mod directories to verify Geyser + Floodgate or Simple Voice Chat statuses.

## System Requirements
- **OS**: Windows 10 (1809+) or Windows 11
- **Architecture**: x64 (64-bit)
- **Framework**: .NET Desktop Runtime 8.0 (automatically managed by the installer)
- **Active Internet**: Required for mod/plugin installs, Adoptium JRE provisioning, Playit.gg agent calls, and cloud backups.
